Frequently Asked Questions

What even is MLU?

MLU is a movie lookup site that uses a movie database api from rapidapi.com. Right now our site is on alpha build and features only looking for movies only. In the future we will be adding features like looking for actors.

Where can I give some feedback on the site?

You should be able to see a button on the navigation bar that allows you to submit a form in which we will be able to recieve that form submission and take in consideration it. We'd apprecaite any feedback to make this site more reliable.

What kind of features you plan on bringing to this site?
